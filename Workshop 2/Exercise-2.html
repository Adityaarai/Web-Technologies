Importance of Alt and Title attributes in image tag:

Alt attribute is used to make the user experience easier for screen readers or blind people since they use braille to visit a website.Alt attribute describes what is in a picture to the reader.Alt attribute cannot be seen on the webpage. Image alt tags should be short for describing the image's purpose.
Although Title attribute describes an image just like alt tag, it is used to make user experience nicer for the user. The text which is used to describe the image just hovers over the picture when the mouse cursor is placed on the image. Image titles can be longer for communication with the user.
